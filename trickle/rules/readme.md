# Règles du Projet

## Maintenance du fichier README
- Ce fichier doit être mis à jour à chaque modification majeure des règles ou de la structure du projet.
- Il sert de référence pour les préférences de l'utilisateur.

## Préférences Utilisateur
- **Avatars** : Ne jamais utiliser le style "Bitmoji" ou "Cartoon" (ex: Dicebear Avataaars). Privilégier les initiales ou des icônes neutres.
- **Mode Sombre** : S'assurer que le mode sombre est fonctionnel à tout moment (configuration Tailwind `darkMode: 'class'`).
- **Base de Données** : Le projet utilise une base de données en ligne persistante (Trickle DB). Ne pas revenir à du LocalStorage.
